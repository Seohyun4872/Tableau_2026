## 23강: 분산형 차트

<!-- 분산형 차트에 관해 배우게 된 점을 적어주세요 -->

분산차트: 파라미터 간 상관관계 확인에 주로 활용

- 행.열 설정 후, 추가적으로 나타내고 싶은 필드를 마크에 추가  
- 추가 필드를 세부정보에 드래그, 색상별로 추가 그분하고 싶다면 색상에도 추가해주고 모양을 다르게 하고 싶다면 마크>모양에서 수정 가능  

>>추세선추가

- 왼쪽 상단>분석탭 : 모델 추세선을 시트에 드래그앤드랍  
- 제품 범주별로 추세선을 볼 수 있음  
- 필드(제품)별로 상관없이 전체 추세선을 보고 싶다면, 시트 추세선에 대고 마우스 오른쪽 클릭 >> 모든 추체선 클릭> 요소에서 범주 해제  

여기서 제품별 추세선과 전체 추세선을 함께 보고싶다면?  

열 선반에서 컨트롤 키를 눌러서 복사  
두 필드 중 하나의 추세선을 우클릭> 모든 추세선 편집> 범주를 활성화  
열 선반에서 이중축 선택   
머릿글 해제해서 정리

## 24강: 히스토그램

<!-- 히스토그램에 관해 배우게 된 점을 적어주세요 -->
모양은 막대랑 비슷하지만, 불연속형을 범위로 그룹화해서 보여준다는 점에서 차이가 있음, 그래서 측정값 만으로 그래프를 그릴 때 주로 사용하는 표현방식 

<히스토그램: 구간차원 생성>

테이블 매출 필드에서 마우스 우클릭 > 구간차원 선택 > 구간크기를 태블로에서 초기에 자체 설정해줌

<img width="1373" height="1107" alt="image" src="https://github.com/user-attachments/assets/d41ece6c-da38-460c-a655-d83ff5f0abe9" />

<최대값, 최솟값이 극단적이어서 확인이 어려운 경우>  
로그값으로 변경: 축에다가 마우스 우클릭 > 축편집 > 눈금에서 로그 활성화   

- 양수: 양수값만 눈금에 표시  
- 대칭: 양수,0,음수 값을 포함하는 데이터를 로그배율 축에 표시  

**히스토그램과 막대그래프의 차이**

- 히스토그램: 연속형 측정값을 통해 수치데이터의 빈도를 표시하는 양적데이터이고, 막대와 막대 사이가 붙어있음  
- 막대그래프: 불연속형 여러 범주 데이터를 비교하기 위해 사용되며, 막대와 막대 사이에 공백이 존재함.

## 29강: 그룹

<!-- 그룹에 관해 배우게 된 점을 적어주세요 -->
수동으로 필드에 있는 항목들 묶기 가능   
기존 데이터 원본에 없는 사용자 지정 그룹 필드 만들기 가능  
 
1. 뷰에서 그룹 만들기  
- 드래그 후, 마우스 우클릭 > 그룹 선택 : 그룹아이콘과 함께 그룹 필드 만들어짐  
2. 항목별로 묶을 필드 선택해 그룹 만들기  
제품이름 필드 우클릭 > 만들기 > 그룹 클릭  
- 마우스 shift 누른 뒤 묶어줄 것들 고르고 그룹 누르기  
- 기타포함 누를 시, 남은것들이 다 기타로 들어가게 됨  
- 이후 만들어진 필드(그룹) 을 마크>색상탭에 넣으면 그룹별로 색상이 구분되어 뷰에서 보임  
- 해당 필드를 열로 바꿔줘도 괜춘

>> 제품이름(그룹)에서 기타를 제외하는 필터 적용(마우스우클릭> 필터> 기타 해제)
